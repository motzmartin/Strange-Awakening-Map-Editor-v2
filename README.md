# Controls  

| Key | Action |
|-----|--------|
| `Z` | Move up |
| `Q` | Move left |
| `S` | Move down |
| `D` | Move right |
| `E` | Select tile |
| `F` | Toggle front mode |
| `R` | Enable remove mode |
| `C` | Enable collision editor mode |
| `X` | Export current map to `levels/level1.dat` |
| `L` | Load `levels/level1.dat` |
| `G` | Enable/disable grid |
